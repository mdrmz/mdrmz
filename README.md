## 🚀 Featured Projects & Systems

### 🛰️ **Autonomous Drone Telemetry System**
Custom Python ground station with real-time SPI communication architecture between Raspberry Pi and Arduino using LoRa modules for long-range data transmission. Includes:
- Live GPS tracking and altitude monitoring
- Sensor data aggregation and analysis
- Custom communication protocol
- Web-based dashboard for telemetry visualization

**Stack:** Python, LoRa, SPI, Raspberry Pi, Arduino

---

### 🚗 **Edge ALPR (Automatic License Plate Recognition)**
Fully autonomous, offline license plate recognition system engineered to run seamlessly on **Raspberry Pi 5**. Features:
- Real-time video processing
- YOLOv8-optimized detection pipeline
- Zero internet dependency
- High accuracy on embedded hardware
- Timestamp and geolocation logging

**Stack:** Python, YOLOv8, OpenCV, Raspberry Pi 5, TensorFlow Lite

---

- Industrial-grade performance and reliability

**Stack:** Python, FastAPI, React, PostgreSQL, Docker, AWS
